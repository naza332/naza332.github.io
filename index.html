Life Log Events System

This was a functional prototype for a lifelog system, where I used the .Net framework with C# and the google maps NuGet package to build a program which loads events from a SOAP XML file and displays them on the map. Additionally a user can add events to the file by (right) clicking on the map and selecting the relevant event type. This was mainly created for people who suffer from dementia.

Generic "doom" shooter prototype

A 2.5D game heavily inspired by classic fps shooters. Built on WebGL so playable within browser. The objective of the game is to reach the end of a timed level while being shot at by fedoramen (to compete with your friends for bragging rights). The game has SFX, music and health/ ammo pickups as well as doors which open through triggers.

2D Space Shooter

A 2D FFA Multiplayer space shooter game with three ammo types. It supports 6 players, the local player will see all other ships as red. It has healthbars and names which is fully networked (using mirror). Tested this game on an AWS Amazon VPS with some friends and it was fun. Eventually moved this project to steam through the fizzysteamworks. It is not on the store but can be played with friends through the development test mode (steamid).
